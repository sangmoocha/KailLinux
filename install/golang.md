```sh
  sudo add-apt-repository ppa:longsleep/golang-backports
  sudo apt update
  sudo apt install golang-go
```

  kali@kali:~$ go env

> GOPATH의 디렉토리는 기본적으로 생성 되지 않는다. 따라서 임의로 디렉토리를 생성해 주어야 한다.
```sh
  cd ~
  madir -p go/workspaces/hello/src
  cd go/workspaces/hello/src
  code .
```
